Date Time

If you do not specify the secondsValue and msValue parameters, the values returned from getSeconds() and getMilliseconds() methods are used. If a parameter you specify is outside of the expected range, setMinutes() attempts to update the date information in the Date object accordingly. For example, if you use 100 for secondsValue, the minutes will be incremented by 1 (minutesValue + 1), and 40 will be used for seconds. Examples
